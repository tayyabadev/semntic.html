HTML Introduction

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content

HTML Tags

  • <br> Used for line break
  • <p> Used for writing Paragraphs
  • <link>Used for add links
  • <image>Used for Add images
  • <aside>Used for add information to side of page
  • <header>Used for main headings
  • <footer>Used for Copyright claim and other
  • <figcaption> Used to show caption along with figure
